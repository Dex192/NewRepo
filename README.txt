Название
Информационная система автоматизации учёта музыкальных инструментов в магазине

Назначение
Программа предназначена для автоматизации процессов учёта, хранения, поставок и продаж музыкальных инструментов.
Система позволяет вести складской учёт, оформлять продажи, формировать отчёты и анализировать данные.

Основные возможности

• Авторизация пользователей
• Добавление, редактирование и удаление товаров
• Учёт поставок и продаж
• Использование справочников (производители, категории, страны)
• Автоматическое формирование чеков
• Формирование отчётов в форматах TXT и Excel
• Отображение статистики в графическом виде
• Мультиязычность (русский / английский)
• Смена темы оформления (светлая / тёмная)
• Резервное копирование базы данных
• Логирование действий пользователей

Используемые технологии

• Python 3
• Tkinter (графический интерфейс)
• SQLite (база данных)
• openpyxl (Excel отчёты)
• matplotlib (графики)
• tkcalendar (выбор дат)

Архитектура проекта

Проект реализован по модульному принципу и включает следующие модули:

• main.py — главный модуль
• database.py — работа с базой данных
• auth.py — авторизация
• instruments_module.py — управление товарами
• sales_module.py — продажи
• supplies_module.py — поставки
• reports_module.py — отчёты
• storage.py — резервное копирование
• languages.py — мультиязычность
• app_logger.py — логирование

База данных

Используется SQLite база данных music_store.db, создаваемая автоматически при первом запуске программы.
